Maven ktlint Plugin

This plugin provides the ability to use ktlint to format and check your source code against the `ktlint` anti-bikeshedding code style.

Goals Overview

  • ktlint:format format your production Kotlin sources using `ktlint`.
  • ktlint:testFormat format your test Kotlin sources using `ktlint`.
  • ktlint:lint lint your production Kotlin sources using `ktlint`.
  • ktlint:testLint lint your test Kotlin sources using `ktlint`.

Usage

General instructions on how to use the ktlint plugin can be found on the usage page.
