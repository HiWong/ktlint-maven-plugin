Usage

Some brief examples on how to use this plugin.

Formatting and linting

Probably the most common use case is to want to both format and lint your code; add the following plugin configuration to your POM.

<build>
  <plugins>
    ...
    <plugin>
      <groupId>com.github.gantsign.maven</groupId>
      <artifactId>ktlint-maven-plugin</artifactId>
      <version>0.9.0-SNAPSHOT</version>
      <executions>
        <execution>
          <id>format-and-lint</id>
          <goals>
            <goal>format</goal>
            <goal>testFormat</goal>
            <goal>lint</goal>
            <goal>testLint</goal>
          </goals>
        </execution>
      </executions>
    </plugin>
    ...
  </plugins>
</build>

Formatting only

If you just want `ktlint` to automatically fix what it can, add the following plugin configuration to your POM.

<build>
  <plugins>
    ...
    <plugin>
      <groupId>com.github.gantsign.maven</groupId>
      <artifactId>ktlint-maven-plugin</artifactId>
      <version>0.9.0-SNAPSHOT</version>
      <executions>
        <execution>
          <id>format</id>
          <goals>
            <goal>format</goal>
            <goal>testFormat</goal>
          </goals>
        </execution>
      </executions>
    </plugin>
    ...
  </plugins>
</build>

To run this from the command line (without adding the plugin to your POM) you can run the following from your console:

mvn com.github.gantsign.maven:ktlint-maven-plugin:0.9.0-SNAPSHOT:format \
    com.github.gantsign.maven:ktlint-maven-plugin:0.9.0-SNAPSHOT:testFormat
